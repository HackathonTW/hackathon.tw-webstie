黑客松 是什麼 ? 可以吃嗎 ?

黑客松（Hackathon）是黑客（Hack）與 馬拉松（Marathon）的複合字
用來表示一個「馬拉松式的黑客活動」

黑客松
不是比賽

一個讓設計人在閒暇時間，參與的開發運動。解放自己，做自己想做的設計和開發，挑戰自己，在有限時間內做出東西

任何人
都可以參加

黑客松是任何人都可以參加的活動，只要你有興趣、想實際參與多人設計和開發，無論你是學生、普通程式人員、還是新手設計師、初級自造者，都歡迎帶著學習、共樂的心前來參加！

自造、學習
與交流

有高手大大、帥哥與美女人才們，有電有網路、有吃的有喝的，既然台灣的夏天嚴熱，當然還有附上空調視野絕佳的舒適環境，讓大家安心動手做設計、做開發！挑戰自我！

即便您還對程式不熟悉

我們亦有一些程式培訓課程，逐步地引領學員進入黑客松的生態。
以下是我們目前提供的課程內容

QML 課程

誰說使用 Qt 進行 UI 的設計，一定要會寫 C++，我們將帶領著各位學員使用 QML，一步步地從無打造自己專屬的 UI 介面。
課程內容

Node.js 課程

我們結合 NodeSchool 工作坊的教材，由導師（Mentor）逐步的帶領著各位學員，進入 Node.js 世界，由淺入深的一步步的探索 Node.js 的神奇之處，自己打造應用程式其實很簡單。

JavaScript 課程

即便你是不曾寫過或還只是剛學不久的初學者，我們提供 JavaScript 程式的基礎教學，帶領著各位學員從最基礎的語法開始學習。只要有心，人人都可以是 Coder。

感謝合作夥伴們的力挺

DOITT

白金級夥伴

DOIT Taiwan共創公域（DOITT），全名為 Delivering Open Innovations for Tomorrow，是為一青年科技創意實踐平台，由經濟部技術處支持，資策會產業情報研究所（MIC）青年領航學院負責執行，是為因應全球軟硬整合創新趨勢下，鼓勵及協助青年將創意實踐為產品或事業的青年平台，DOIT Taiwan 信仰開放式創新，一方面匯聚青年創意點子，協助媒合資源實踐，另一方面則希望影響產業擁抱開放，達成創新目的。

AvengerGear Inc.

白金級夥伴

人本產品設計：以人為本，從零開始人生無法從來，但產品可以。
我們尋找一種以人為本、在生理與心理上永續生活的最佳方法。我們選擇了「產品製造」為出發點，因為我們發現在產品上，可有效的藴含意念、蘊含文化與品味，還有那些我們心靈所嚮往的一切價值；並也在我們在日常生活中，供給我們所需求一切的實質幫助與滿足。我們除去一切刻板制約，從零開始思考、從最原始來打造。我們不怕犯錯，因為我們相信錯誤是演化的基石、是最佳化的依據；並也是設計產品最完美有效的方法。

Qt

白金級夥伴

Qt 是領先的面向 桌面、嵌入式和移動作業系統的跨平台應用程式與UI開發框架。Qt使用標準C++語言，被廣泛用於開發包含GUI的軟體應用程式和非 GUI 應用程式，以提供檔案處理、數據庫訪問、XML解析、線程管理和網路支援等功能。Qt具備強大的全框架功能，支援構建高性能原生應用程式，以及進行混合開發，允許開發人員選擇能夠帶來最佳用戶體驗的工具。 The Qt Company 是芬蘭商Digia Plc完全擁有的子公司，負責所有Qt相關活動，包括產品開發、商業許可、開源許可，以及在開放管理模式下開展的 Qt Project 。

卡市達

白金級夥伴

卡市達願意是創業者的家，是你與人事物之間溫暖而親近的共享空間，是更多實際的、虛擬的平台產生的源頭。 卡市達創業加油站成立於2010年十月十日，一個致力於推動台灣創業風氣的育成中心，在塵囂中因夢想而存在，為創造國際永續創意生態系而努力。珍惜並尊重人與其所有的核心特質是我們的根本價值，提供與自然和諧共生和符合人性的創新是我們一貫的脈絡、速度，水平思考與扁平化是我們的處事態度。我們以身做則並自許成為榜樣，以行動為自己創造更好的生活，以熱情和愛來為更大的自己服務。

Skywatch

白金級夥伴

隨著頻寬改善、智慧型手機普及、網路攝影機在系統以及晶片端的漸趨成熟以及雲端服務的興起， 遠距雲端影像技術已漸漸深入中小企業以及家庭，帶來有別以往的應用價值。 這些新應用能延伸關懷、降低焦慮，使人人都有辦法和自己關心的事件連結在一起。 伴隨雲端影像的時代來臨，行品股份有限公司（Skywatch Innovation）創立於 2011年 11月，旨在帶給消費者方便、易用、可靠、安全的雲端影像產品，並以作為客戶與使用者的長期合作夥伴為目標。
